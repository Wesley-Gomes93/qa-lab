# Imagens do portfolio

Adicione aqui o screenshot da pipeline (GitHub Actions) passando.

**Arquivo esperado:** `pipeline-verde.png`

Como obter:
1. Acesse https://github.com/Wesley-Gomes93/qa-lab/actions
2. Abra um run que passou (ícone verde)
3. Faça um print da tela mostrando os jobs (lint, build, tests, e2e, report)
4. Salve como `pipeline-verde.png` nesta pasta

Se o arquivo não existir, o portfolio exibirá uma mensagem alternativa.
