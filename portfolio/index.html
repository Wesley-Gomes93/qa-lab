MCP Lab Agent

Agente MCP universal para Cursor IDE: detecta automaticamente frameworks de teste (E2E, Unit, Mobile, API, Python), estrutura de projetos e executa QA workflows sem configuração manual.

  • Suporte universal: Cypress, Playwright, WebdriverIO, Jest, Vitest, Mocha, Appium, Detox, Robot Framework, pytest
  • Execução de testes (backend, frontend, mobile, API), geração de casos, análise de falhas e sugestões de correção
  • Linter, coverage, bug reports e templates de teste
  • Zero config: use com npx -y github:Wesley-Gomes93/qa-lab-agent-mcp
