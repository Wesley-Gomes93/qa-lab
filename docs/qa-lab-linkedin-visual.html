QA Lab – Como tudo se desenrola

Full-stack + testes automatizados + agente com menu interativo (MCP)

Arquitetura

Frontend
Next.js
Backend
Express
PostgreSQL
Cypress
E2E + API
MCP Server
run_tests
Agente
Menu

Você pode rodar os testes de duas formas: manual (terminal + Cypress) ou pelo agente, que se conecta ao MCP e abre um menu para você escolher qual suíte ou qual arquivo executar.

Como rodar?

1
Subir o banco database/
docker-compose up -d
2
Subir o backend backend/
npm install && npm run dev
3
Subir o frontend frontend/
npm install && npm run dev
4
Rodar os testes tests/
npm test

Ou uma suíte só: npx cypress run --spec "cypress/e2e/admin/**/*.cy.js"

1
Entrar na pasta do agente e instalar
cd agents && npm install
2
Abrir o menu interativo
npm run agent:run-tests
[agent] Conectado ao MCP server qa-lab-mcp. O que você quer rodar? Leia o que cada opção faz e digite o número da opção desejada. ═══════════════════════════════════════════════════════════ 1. Todos os testes → Roda a suíte completa: auth, api, admin e ui (pode demorar mais). 2. Admin – painel do administrador → Testes do dashboard admin: editar idade, validação 18–80, inativo, filtro, exclusão. 3. Auth – login, logout e registro → Login admin, logout, registro (título, combinações, fluxo completo). 4. API – chamadas diretas ao backend 5. UI – elementos da tela inicial 6. Rodar apenas um arquivo de teste → Lista cada arquivo .cy.js; você escolhe um e só esse é executado. 0. Sair (não roda nenhum teste) ─────────────────────────────────────────────────────────── Digite o número da opção e pressione Enter: _
3
Digite o número Ex.: 2 = só testes Admin, 6 = depois escolhe um arquivo .cy.js. O agente chama o MCP, o MCP roda o Cypress e devolve o resultado.

Resumo

Manual: você sobe cada parte (banco, backend, frontend) e roda npm test (ou um --spec) na pasta tests/.

Pelo agente: um comando (npm run agent:run-tests), menu com todas as opções de teste, você escolhe por número e o agente + MCP executam o Cypress e devolvem o resultado. Tudo integrado.
