# Nuclei

[Project Nuclei](https://github.com/projectdiscovery/nuclei) — scanner baseado em templates para vulnerabilidades conhecidas.

## Uso local

```bash
# Instalar: go install -v github.com/projectdiscovery/nuclei/v2/cmd/nuclei@latest
nuclei -u http://localhost:3000
```

## Integração

Ver `../../scripts/run-security-scan.sh` para execução automatizada.
