Vulnerability endpoints

  • /search?q= — Reflected XSS (try: <script>alert(1)</script>)
  • /comments — Stored XSS (submit a comment with script tag)
  • /redirect?url= — Open Redirect
  • /debug — Information Disclosure
  • /dom-xss.html — DOM-based XSS
  • /file?name= — Path Traversal (try: ../..)
